h1. Ruby Wrapper for Namecheap.com API

Allow the ability to check domain availability through namecheap.com API.

h2. Usage

In your Gemfile:
<pre>
  <code>
gem 'namecheap', :git => 'git://github.com/Agiley/namecheap.git'
  </code>
</pre>

Generate namecheap config file:

<pre>
  <code>
rails generate namecheap
  </code>
</pre>

Use the client:

<pre>
  <code>
namecheap = Namecheap::Client.new
namecheap.is_domain_available? 'hashrocket.com' # => false
  </code>
</pre>
